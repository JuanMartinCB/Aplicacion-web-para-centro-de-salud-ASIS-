# Proyecto ASIS
Aplicación web para gestión integral de un centro de salud.

*Arquitectura MVC con separación clara de capas
*Persistencia de datos mediante Hibernate/JPA
*Sistema de autenticación con gestión de roles (administrador, personal_salud, referente_org_social)
*Geolocalización mediante integración de la API de Leaflet
*API REST para comunicación frontend-backend
*Documentación y testeo de API con Swagger

Backend:
*Java
*Apache Tomcat (servidor de aplicaciones)
*Hibernate/JPA (ORM)
*Maven (gestión de dependencias)

Frontend:
*Angular
*TypeScript

Proyecto final desarrollado para la materia Java y Aplicaciones Avanzadas - Ingeniería en Computación, UNLP (2025)
